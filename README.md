# 🃏 三国杀助手 - Sanguosha Assistant

> 一款离线可用的三国杀武将/卡牌/规则查询 Android 应用，基于 WebView PWA 技术构建。

[![Gitee](https://img.shields.io/badge/Gitee-cpufreestyle%2Fsanguosha--assistant-red?logo=gitee)](https://gitee.com/cpufreestyle/sanguosha-assistant)
[![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Android-green?logo=android)]()

## 📱 功能特性

| 模块 | 说明 |
|------|------|
| 🔍 **武将查询** | 蜀/魏/吴/群四大势力，10位经典武将，含技能详解、体力值 |
| 🃏 **卡牌查询** | 22张标准版卡牌，含杀、闪、桃、锦囊、装备等 |
| 📖 **游戏规则** | 出牌规则、距离计算、伤害流程、判定阶段等核心规则 |
| ❓ **智能问答** | AI 回答游戏相关问题，支持自然语言提问 |

### 🌟 特色亮点

- **完全离线** - 无需网络，随时随地查询
- **极速响应** - 本地数据，毫秒级加载
- **轻量体积** - APK 仅 83KB，几乎不占空间
- **简洁界面** - 移动端优先设计，操作流畅

## 🔧 本地构建 APK

### 环境要求

| 依赖 | 版本 | 说明 |
|------|------|------|
| Python | 3.8+ | 图标生成脚本 |
| Pillow | 9.0+ | Python 图像库 |
| JDK | 17+ | Java 编译环境 |
| Android SDK | 34+ | build-tools, platform |

### 构建步骤

```powershell
# 1. 克隆项目
git clone https://gitee.com/cpufreestyle/sanguosha-assistant.git
cd sanguosha-assistant/sanguosha-mobile

# 2. 安装 Python 依赖（可选，仅用于生成图标）
pip install Pillow

# 3. 生成应用图标（可选）
python make_icons.py

# 4. 运行 APK 构建脚本
python build_apk.py
```

APK 输出位置：`dist/三国杀助手.apk`

### Android SDK 安装

如未安装 Android SDK：

```powershell
# 下载 command-line tools
# https://developer.android.com/studio#command-tools

# 解压到 D:\Android\cmdline-tools\latest\

# 设置环境变量
$env:ANDROID_HOME = "D:\Android"
$env:PATH += ";$env:ANDROID_HOME\cmdline-tools\latest\bin"

# 安装必要组件
sdkmanager --install "platforms;android-34"
sdkmanager --install "build-tools;34.0.0"
```

## 📖 游戏规则速查

### 距离计算

```
实际距离 = 基础距离 - 进攻马(+1) + 防御马(-1)
基础距离 = 相邻为1，隔一个为2，以此类推

攻击范围 = 武器距离 - 进攻马(+1)
无武器时，攻击范围默认为1
```

### 出牌流程

```
1. 准备阶段 → 判定生效
2. 判定阶段 → 触发判定技能
3. 摸牌阶段 → 摸2张牌
4. 出牌阶段 → 使用任意牌
5. 弃牌阶段 → 弃置多余手牌
6. 结束阶段 → 触发结束技能
```

### 伤害流程

```
使用牌 → 指定目标 → 目标响应 → 结算伤害
         ↓
    计算距离 → 检查范围 → 使用防具
```

## ❓ 常见问题

**Q: 这款应用需要网络吗？**  
A: 完全不需要。应用内所有数据已打包，离线可用。

**Q: 为什么安装时提示"未知来源"？**  
A: APK 未上架应用商店，需手动允许安装权限。仅此一次。

**Q: 支持哪些 Android 版本？**  
A: Android 5.0+（API 21+）均可运行。

**Q: 武将数据会更新吗？**  
A: 当前为标准版数据，后续会扩展更多武将。
